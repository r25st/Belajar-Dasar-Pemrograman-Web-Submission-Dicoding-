Dasar html

HTML adalah singkatan dari Hypertext Markup Language, HTML merupakan salah satu bahasa pengkodean atau pemograman yang digunakan untuk membuat halaman website yang ditampilkan pada web browser. Sebagian besar halaman yang kamu temukan pada internet kebanyakan menggunakan Bahasa HTML.



Kerangka HTML ( semantik )

Kerangka dasar HTML semantik adalah struktur kode HTML yang menggunakan elemen-elemen semantik untuk meningkatkan keterbacaan dan aksesibilitas dari sebuah halaman web. Elemen-elemen semantik memberikan makna yang jelas tentang konten yang dikandungnya, sehingga lebih mudah dipahami oleh mesin pencari dan pembaca layar. Berikut adalah penjelasan singkat mengenai beberapa elemen semantik utama dalam HTML :

  1. < header > : Digunakan untuk menampung konten pengantar atau navigasi utama dari sebuah halaman atau bagian dokumen. Biasanya berisi judul, logo, dan menu navigasi.
  2. < nav > : Menyatakan bagian navigasi dari sebuah halaman. Biasanya berisi tautan-tautan ke bagian utama dari situs web.
  3. < main > : Digunakan untuk menampung konten utama dari dokumen, yang unik dan relevan dengan halaman tersebut. Hanya boleh ada satu elemen < main > per halaman.
  4. < article > : Menyatakan konten yang bisa berdiri sendiri dan memiliki makna lengkap jika diambil terpisah. Contohnya seperti artikel berita, posting blog, atau komentar.
  5. < section > : Digunakan untuk mengelompokkan konten tematik atau bagian dari dokumen, seperti bab dalam buku atau bagian dalam laporan.
  6. < aside > : Menyatakan konten yang terkait secara tidak langsung dengan konten utama. Biasanya digunakan untuk sidebar, catatan kaki, atau referensi tambahan.
  7. < footer > : Menyatakan bagian footer dari dokumen atau bagian dari dokumen. Biasanya berisi informasi hak cipta, penulis, tautan ke kebijakan privasi, atau informasi kontak.



css





Dasar css

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web. Berikut adalah penjelasan dasar tentang CSS:

  1. Selektor dan Properti:
    • Selektor: Menentukan elemen HTML yang akan diberikan gaya. Contoh: p, .class, #id.
    • Properti: Menentukan jenis gaya yang akan diterapkan. Contoh: color, font-size, margin.
    • Nilai: Menentukan nilai dari properti. Contoh: red, 16px, 10px.

  2. Cascading (Penurunan):
    • CSS menggunakan hirarki atau urutan untuk menentukan gaya yang diterapkan jika ada konflik. Urutan prioritas: inline styles, internal styles, external styles.

  3. Spesifisitas:
    • Spesifisitas menentukan gaya mana yang lebih prioritas jika ada beberapa gaya yang diterapkan ke elemen yang sama. Urutannya: ID (#id), kelas (.class), dan elemen (element).

  4. Box Model:
    • Setiap elemen dianggap sebagai kotak yang terdiri dari margin, border, padding, dan content.

  5. Layouting:
    • Menggunakan properti seperti display, position, float, dan flexbox untuk mengatur tata letak elemen.

  6. Responsiveness:
    • Menggunakan media queries untuk membuat desain yang responsif terhadap berbagai ukuran layar.

Dengan memahami dasar-dasar ini, Anda dapat mulai mengembangkan dan mengatur tampilan halaman web dengan efektif menggunakan CSS.



Flexbox

Flexbox (Flexible Box Layout) adalah model tata letak CSS yang dirancang untuk mengatur dan menyelaraskan item di dalam wadah secara efisien, baik dalam arah horizontal maupun vertikal. Berikut adalah penjelasan dasar tentang Flexbox:

  1. Flex Container dan Flex Items:
    • Flex Container: Elemen yang memiliki properti display: flex; atau display: inline-flex.
    • Flex Items: Elemen anak langsung dari flex container.

  2. Arah (Direction):
    • Properti flex-direction mengatur arah utama flex container. Nilai umum:
      • row: Default, arah horizontal dari kiri ke kanan
      • row-reverse: Arah horizontal dari kanan ke kiri.
      • column: Arah vertikal dari atas ke bawah.
      • column-reverse: Arah vertikal dari bawah ke atas.

  3. Pembungkusan (Wrapping):
    • Properti flex-wrap mengatur apakah item dibungkus jika tidak muat dalam satu baris atau kolom. Nilai umum:
      • nowrap: Default, tidak membungkus.
      • wrap: Membungkus ke baris atau kolom berikutnya.
      • wrap-reverse: Membungkus ke baris atau kolom sebelumnya.

  4. Justify Content:
    • Properti justify-content mengatur perataan item di sepanjang arah utama. Nilai umum:
      • flex-start: Default, item diposisikan di awal
      • center: Item diposisikan di tengah.
      • flex-end: Item diposisikan di akhir.
      • space-between: Ruang di antara item sama.
      • space-around: Ruang di sekitar item sama.

  5. Align Items:
    • Properti align-items mengatur perataan item di sepanjang arah silang (perpendicular). Nilai umum:
      • stretch: Default, item direntangkan mengisi container.
      • flex-start: Item diposisikan di awal.
      • center: Item diposisikan di tengah.
      • flex-end: Item diposisikan di akhir.

  6. Align Self:
    • Properti align-self mengatur perataan individual item, menimpa align-items pada container.

  7. Flex Grow, Shrink, dan Basis:
    • flex-grow: Mengatur kemampuan item untuk tumbuh.
    • flex-shrink: Mengatur kemampuan item untuk menyusut.
    • flex-basis: Mengatur ukuran dasar item sebelum ruang tambahan dialokasikan.

Dengan memahami dasar-dasar Flexbox ini, Anda dapat membuat tata letak yang fleksibel dan responsif dengan lebih mudah dan efisien.
